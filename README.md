# Space-ML-Analytics

This repository aggregates a specialized collection of Data Science and Machine Learning projects focused on space exploration, astrophysics, and aerospace analytics using Python.

## 🌌 Portfolio Overview

### 🪐 1. ExoPlanet Discovery & Classification
* **Directory:** `01-Exoplanet-Classification/`
* **Objective:** Classify Kepler Objects of Interest (KOI) into confirmed planets or false positives.
* **Tech Stack:** L1 Logistic Regression, Decision Trees, Random Forest, Partial Dependence Plots (PDP).
* **Key Result:** Achieved a peak **0.94 ROC-AUC score** using Random Forest and mapped non-linear planetary radius thresholds.

### 🛰️ 2. Satellite Lifetime Estimation (RSF)
* **Directory:** `02-Satellite-Reliability-RSF/`
* **Objective:** Predict time-to-failure events and reliability curves for satellite constellations.
* **Tech Stack:** Random Survival Forests (RSF), Survival Analysis, Censored Data Processing.
* **Key Result:** Built predictive maintenance curves to minimize orbital equipment downtime.

### ☀️ 3. Solar Wind Anomaly Detection
* **Directory:** `03-Solar-Wind-Anomaly-Detection/`
* **Objective:** Detect space weather anomalies and extreme solar wind events affecting Earth's magnetosphere.
* **Tech Stack:** Time-Series Anomaly Detection (Isolation Forest / Autoencoders / LSTM).
* **Key Result:** Created an early-warning filtering mechanism for dangerous geomagnetic shifts.
